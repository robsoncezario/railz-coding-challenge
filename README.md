# railz-coding-challenge

Railz coding challenge for the Full-stack SWE role.

## Documentation

```
/api-docs
```

## Frontend

```
React JS
Redux
Google Maps
Webpack
```

# Setup

```
cd frontend
Create a .env file and populate the enviroment variables
run npm run install
```

## Run

```
npm run dev
```

## Front-end enviroment variables

```
REACT_APP_API_URL
GOOGLE_MAP_API_KEY
```
